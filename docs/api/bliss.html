bliss.sleep module¶

Implements sleep-phase training of BLISS models.

This module contains the SleepPhase class, which implements the sleep-phase training using pytorch-lightning framework. Users should use this class to construct the sleep-phase model.

class bliss.sleep.SleepPhase(encoder: dict, prior: dict, decoder: dict, annotate_probs: bool = False, slack=1.0, optimizer_params: Optional[dict] = None)[source]¶

Summary line.

Implementation of sleep-phase training using pytorch-lightning framework.

Example

In python script, set up the sleep-phase model and pytorch-lightning trainer:

import pytorch_lightning as pl
from bliss.sleep import SleepPhase

model = SleepPhase(**cfg.model.kwargs)
trainer = pl.Trainer()
trainer.fit(model, data=dataset)
