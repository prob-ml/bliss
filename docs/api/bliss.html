bliss.sleep module¶

Summary for sleep.py

This module contains SleepPhase class, which implements the sleep-phase traning using pytorch-lightning framework. Users should use this class to construct the sleep-phase model.

class bliss.sleep.SleepPhase(encoder_kwargs: dict, decoder_kwargs: dict, annotate_probs: bool = False, optimizer_params: Optional[dict] = None)[source]¶

Summary line.

Implementation of sleep-phase training using pytorch-lightning framework.

Parameters

cfg (DictConfig) – OmegaConf configuration from YAML files

Example

In python script, set up the sleep-phase model and pytorch-lightning trainer:

import pytorch_lightning as pl
from bliss.sleep import SleepPhase


model = SleepPhase(cfg)
trainer = pl.Trainer()
trainer.fit(model, data=dataset)
